## 📝 Task Description
Perform **Exploratory Data Analysis (EDA)** on a marketing campaign dataset.

**Objective:** Instead of building a Machine Learning model, focus on the **"Why"**. Clean the data, visualize the funnel, and write a **1-page summary report** recommending which marketing channels should receive more budget based on **ROI**.

---

## ✅ What I Did — 8 Step EDA Process

| Step | Action |
|------|--------|
| 1 | Load marketing campaign dataset (1000 rows) |
| 2 | Data cleaning — remove invalid rows, cap outliers |
| 3 | Overall performance summary |
| 4 | Channel performance analysis (ROI, CTR, Conversions) |
| 5 | Campaign type analysis |
| 6 | Region & Age group analysis |
| 7 | Key business insights & recommendations |
| 8 | Generate visual report (8 charts) |

---

## 📊 Key Findings

### 🏆 Channel Rankings by ROI
| Rank | Channel | Avg ROI |
|------|---------|---------|
| 1 | Email | 626% |
| 2 | SMS | 305% |
| 3 | Google Ads | 10% |
| 4 | Social Media | 6% |
| 5 | Influencer | -66% |

### 💡 5 Business Recommendations
1. **Boost Email & SMS** budget by 20-30% — highest ROI
2. **Focus on 35-44 age group** — highest conversion rate
3. **Prioritize South region** campaigns
4. **Run more Brand Awareness** campaigns
5. **Cut Influencer spending** — negative ROI
